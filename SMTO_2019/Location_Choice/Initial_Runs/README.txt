This folder contains work our initial model runs for the 2019 SMTO location choice problem.
We estimate new parameters using the same model foormulation.
We also use the parameters obtained from the 2015 SMTO location choice model, implementing this in both Biogeme (Python) and mlogit (R).

Code files:
- 2015_Parameters.R: Calculation of metrics for 2019 data with 2015 parameters
- 2015_Parameters_Metrics (outdated).ipynb
- Initial_Runs.R (outdated): Singly- and doubly-constrained gravity models (i.e. with/without ASCs) with/without closest dummies
- Updated.R: Estimation of reference 2019 model

Results files:
- Results.xlsx: Parameters and metrics from 2015 parameters and initial runs
