This folder contains our work on developing a mode choice model for the 2019 SMTO dataset.

Code files (outdated):
- Basic_Model.ipynb: Biogeme implementation of basic mode choice model with and without cost
- Basic_Model.R: R implementation of basic mode choice model

Results files:
- Base_Model.html: Biogeme output for basic mode choice model
- Cost.html: Biogeme output for basic mode choice model with costs
- 2015_2019_Comparison.xlsx: A comparison of the estimated parameters and metrics for the basic 2015 and 2019 mode choice models
