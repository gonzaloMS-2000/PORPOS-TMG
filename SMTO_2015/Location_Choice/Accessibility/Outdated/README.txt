This folder contains outdated work on accessibility with R.

Code file:
- Create_Accessibility_Data.ipynb: Used to create MC_Input.csv
- Segmented_Accessibility.R: Run segmented models and print results to file

Created data files:
- MC_Input.csv: Input for accessibility model including mode choice and location choice columns
- Accessibility_Input.csv: Input to accessibility model with manually-calculated logsums

Results files;
- Segmented_Results.xlsx: Results for accessibility, distance, and accessibility+distance models by segment
