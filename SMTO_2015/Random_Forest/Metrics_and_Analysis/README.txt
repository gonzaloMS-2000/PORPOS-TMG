This folder includes useful code for analyzing model results.

Code files:
- Confusion_Matrox_and_Metrics.ipynb: Sample code for generating and showing confusion matrices, as well as calculating metrics such as precision, recall, and Matthews.
- Tree_Example.ipynb: Sample code for showing a tree from the forest in the notebook

Other files:
- tree.dot: File storing tree, for Tree_Example.ipynb
- tree.png: File to visualize tree created from tree.dot, for Tree_Example.ipynb
