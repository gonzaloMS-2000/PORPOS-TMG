This folder contains our work on including socioeconomic variables in the 2015 SMTO mode choice model.

Code files:
- Socioeconomic_Variable_Runs.R (outdated): Run basic mode choice model with socioeconomic variable and print results to file
- Selected.R: Implementation of the model with selected variables.

Results files:
- All_Outputs.xlsx: Various variable combination results from Socioeconomic_Variable_Runs.R summarized in one file.
- Selected_Results.xlsx: Summary of estimates and metrics for selected model.
