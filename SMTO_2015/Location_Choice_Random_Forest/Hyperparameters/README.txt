This folder includes our work on hyperparmeter tuning for the Random Forest model.

Code files:
- Hyperparameter_Tuning.ipynb: Max depth, number of estimators, min leaf size
- Randomized_and_Grid_Search.ipynb: Randomized and grid search with cross-validation for hyperparameter tuning
- Weighting_Systems.ipynb: Comparing weights by class, weights by sample (expansion factors) to unweighted model

Results files:
- Weighting_Systems_Results.csv: Results from Weighting_Systems.ipynb
