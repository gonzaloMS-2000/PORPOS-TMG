This folder contains our proposed location choice model for location choice.
The full model includes several campus attributes that were removed for the simple model where these variables are unavailable (i.e. SMTO 2019)

Code files:
- Full_Proposed.R
- Simple_Proposed.R

Results files:
- Full_Results.xlsx: Parameters, metrics, and confusion matrices for the full proposed model.
	Note that the proposed model is the distance model as it is much simpler than the combined model, yet nearly as effective.
