This folder includes our results from experimenting with campus-specific variables using mlogit in R.

Code files:
- Campus_Vars.R: Generate Initial_Results.xlsx
- Campus_Vars_Interacted.R: Interact campus variables with socioeconomic variables
	Family:Domestic enrollment rate found to be significant

Results files:
- Outdated.xlsx: Outdated results with campus variables interacted with socioeconomic variables
- Initial_Results.xlsx: Initial results with all campus attributes.
	Tuition was found to be insigifnicant so is analyzed further in Tuition_Results.xlsx
- Tuition_Results.xlsx: Significances when tuition is interacted with living arrangement
- Selected_Results.xlsx: Results for distance, accessibility, and combined models with selected campus attributes
