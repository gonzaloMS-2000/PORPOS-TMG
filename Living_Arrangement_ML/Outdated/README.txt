This folder contains our original files from our work with Model Builder.

Data files:
- SMTO_2015_ML_Format.csv: Model Builder input for 2015 SMTO data
- TTS_2016_ML_Output.csv: Model Builder output for LightGbmBinary model

Notebooks:
- TTS_2016_Output_Tabulation.ipynb: Tabulating classifier results, comparing predicted TTS living arrangements with SMTO patterns

The Model Builder folder contains C# code which can be loaded in Visual Studio.
